# Тестовое задание с регистрацией, таблицей продуктов, валидациями, формами и сортировками

## Установка

Сначала нужно устовить зависимости:

```bash
yarn install
```

Потом запустить сервер. Будет доступен по [http://localhost:3000](http://localhost:3000).

```bash
yarn run dev
```
